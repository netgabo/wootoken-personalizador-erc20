Un plugin que sirve para agregar una nueva moneda para el soporte ERC20.

== Description ==
Este complemento agregará una nueva moneda personalizada para ayudarlo a recibir en su sitio web WooCommerce tokens personalizados compatibles con la máquina virtual de ethereum (ERC20).

== Instalación ==
1. Descargar e Instalar WooCommerce
2. Descargar el complemento
3. Instalar el complemento
4. Configure su propia moneda en las opciones.

== Preguntas frecuentes ==
1. Cómo cambiar moneda
R: Debe cambiar la moneda en la configuración de WooCommerce
